# INSIDE 프로젝트 PRD

## 1. 프로젝트 개요

### 프로젝트명
INSIDE – 새로운 세대의 소셜 네트워크

### 목적
- Instagram UI/UX 벤치마크 수준의 경험 제공
- Next.js + Supabase + Clerk 스택 실무 맞춤 실습
- 2~3일 내 구현 가능한 MVP 설계, 확장 가능한 아키텍처
- 인싸이드만의 "+알파 기능"으로 차별화: 나만의 공간(스토리/공개범위 지정), 트렌드 큐레이션(추천 피드), 스팟 태그

### 기술 스택
- 프론트: Next.js 14, TypeScript, Tailwind CSS
- 백엔드: Supabase (DB+Storage), Clerk (OAuth/이메일 인증 등)
- 배포/운영: Vercel, Supabase 프로젝트, Clerk 관리 콘솔
- 보안: API Routes 내 사용자 인증 → 자체 보안 미들웨어, DB Row Level Security는 미사용(RLS 비활성)

## 4. MVP 핵심 기능
1. **클릭 한 번 회원가입/로그인(Clerk)**
2. **홈 피드(최신순, 무한스크롤)**
3. **포스트 업로드(이미지1, 캡션, 업로드/유효성)**
4. **좋아요(하트, 더블탭), 실시간 카운트 반영**
5. **댓글(작성/삭제, 2줄 미리보기와 상세목록)**
6. **프로필(게시물 그리드, 팔로우/팔로잉 카운트, 나/타인 분기)**
7. **팔로우/언팔로우(즉시 UI 반영, API 직결)**
8. **적절한 애니메이션(클릭, 스켈레톤, shimmer)**
9. **반응형/디자인 시스템(컬러·폰트·간격 기준화)**


#### (+) INSIDE만의 추천 +알파 예시(2차 우선순위, 설계만)
- 게시글 공개범위(전체/친구/나만)
- 트렌드(실시간 인기/키워드)
- 스팟(위치·모임 태그 등)

## 5. 상세 설계(요약)
- Sidebar: Lucide React 아이콘, Active/hover, 명확한 현재 위치 강조
- PostCard: Header(아바타/닉/시간/메뉴), 정사각 이미지, 하트·댓글아이콘(상태에 따라 변화), 캡션(2줄 초과 시 더보기), 댓글미리보기
- 프로필: 큰 아바타, 팔로우/팔로잉 카운트, 3열 그리드, Hover 시 좋아요/댓글수 표시
- 모달: 상세보기(이미지 50%+댓글 50%, 댓글리스트 스크롤)
- 컬러 스키마/폰트 체계: Instagram 기반, 색상·폰트 상수화(Tailwind 커스텀)
