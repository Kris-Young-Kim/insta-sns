## 테스트 방법

1. **Chrome DevTools 사용**
   - F12 → Device Toolbar (Ctrl+Shift+M)
   - 다양한 디바이스 크기로 테스트

2. **실제 디바이스 테스트**
   - 모바일: iPhone, Android
   - 태블릿: iPad
   - 데스크톱: 다양한 해상도

3. **네트워크 시뮬레이션**
   - Chrome DevTools → Network → Throttling
   - Slow 3G, Fast 3G 등으로 테스트

4. **오프라인 테스트**
   - Chrome DevTools → Network → Offline
   - 오프라인 상태에서의 동작 확인
